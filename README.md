# mysql

### Install mysql repo

```bash
yum localinstall https://dev.mysql.com/get/mysql57-community-release-el7-11.noarch.rpm
yum install mysql-community-server
```

### Basics queries

```bash
show databases;

 USE `database`;
 SHOW TABLES;

 SHOW TABLES FROM `database`;
 
 show fields  from table;
 show columns from table;
 ```
